O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação utilizada para criar páginas web. Ela permite estruturar o conteúdo e a apresentação de uma página na web.

Estrutura Básica de um Documento HTML

Um documento HTML é composto por uma série de elementos e tags que definem o conteúdo da página. Abaixo, temos um exemplo de uma estrutura básica de um documento HTML:

Principais Tags HTML

  • <html>: Define o início do documento HTML.
  • <head>: Contém metadados, como o título e a codificação de caracteres.
  • <body>: Contém o conteúdo visível da página, como textos, imagens e links.
  • <h1>, <h2>, etc.: São usadas para criar cabeçalhos e subcabeçalhos.
  • <p>: Define um parágrafo de texto.

Com o HTML, podemos criar páginas web dinâmicas, interativas e visuais. Ele serve como a base para o desenvolvimento de qualquer site.
